Edusity - Education Website

Edusity is a responsive education website built using React.js. The project showcases a modern university/education platform with smooth navigation, video integration, and a contact form powered by Web3Forms.

# Features

- Responsive design for mobile, tablet, and desktop
- Modern and clean UI
- Smooth scrolling navigation
- Hero section
- About section
- Programs/Courses section
- Campus section
- Testimonials
- Video player popup
- Contact form using Web3Forms
- Footer with social links

# Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Web3Forms
- Git & GitHub
